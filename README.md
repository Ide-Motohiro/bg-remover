# 透過職人 — PNG背景透過 & 境界線補正ツール

## 概要

ブラウザ上で動作するPNG画像の背景透過・アルファ境界クリーンアップ・色かぶり除去ツール。

## 作った背景 / 動機

画像の背景透過は既存ツールでも可能だが、透過後の境界品質に課題が残ることが多い。特に髪の毛のような細かい被写体では、背景色がエッジに混ざり込んで不自然な色になったり、半透明ピクセルがぼやけて残ったりする問題がある。このツールは「透過して終わり」ではなく、透過後の境界を丁寧に仕上げるところまでを一貫して行えることを目指して制作した。

## 主な機能

- **背景透過（色指定 / 明るさモード）** — 指定した色またはしきい値ベースで背景を透明化。スポイトで画像上の色を直接取得可能
- **境界線クリーンアップ** — アルファ二値化・モルフォロジー演算（収縮/膨張）・フリンジ除去の3段階で、透過後のアルファ境界をシャープに整理
- **色かぶり除去** — アルファ合成の逆算により、背景色が混ざり込んだ境界ピクセルから本来の前景色を復元。髪と背景が混じる領域などに有効
- **境界カラー補正** — 境界付近の色を内部参照色に基づいて均一化、または手動で指定した色に統一
- **元に戻す / リセット / 元画像比較** — 操作履歴の管理と結果の確認が随時可能

## 技術的な工夫

### アルファ境界のモルフォロジー演算

境界線クリーンアップではアルファマスクに対してerode（収縮）/ dilate（膨張）を実装した。収縮は「隣接に透明ピクセルがあれば自身も透明にする」を反復し、膨張は逆に周囲の不透明ピクセルの平均色で拡張する。膨張時に色を補完することで、広げた部分が不自然にならないよう配慮した。

### フリンジ除去の二重フィルタリング

エッジピクセルの色を内部の色で置き換える際、単に「隣の不透明ピクセル」ではなく「それ自身もエッジでない完全内部のピクセル」を参照元にしている。これにより、背景色に汚染されたエッジ同士が互いを参照して汚染が伝播する問題を防いでいる。

### アルファ合成の逆算による色復元

色かぶり除去は `表示色 = 前景色 × α + 背景色 × (1 - α)` の式を逆転し、前景色を `(表示色 - 背景色 × (1 - α)) / α` で推定する。完全不透明なエッジピクセルに対しては、背景色との色距離と内部参照色からの色距離を比較して汚染度を推定し、擬似的な実効アルファ値を算出することで逆算を適用している。

### 距離マップによる処理深さの制御

境界処理の各機能では、アルファ境界からのピクセル距離マップを構築し、境界に近いほど強く・遠いほど弱く処理を適用するグラデーション制御を行っている。これにより、内部の本来の色を壊さず境界付近だけを自然に補正できる。

## 使用技術

- HTML / CSS / JavaScript（バニラ、ライブラリ依存なし）
- Canvas API（ImageData によるピクセル単位の画像操作）
- Blob API / Object URL（ダウンロード機能）

## 使い方

1. `bg-remover.html` をブラウザで開く（Chrome推奨）
2. 画像をドラッグ&ドロップまたはクリックで読み込む
3. **背景透過**：色指定モードまたは明るさモードで背景を透明化
4. **色かぶり除去**：混ざっている背景色を指定して境界の色を復元
5. **境界線クリーンアップ**：アルファしきい値と収縮/膨張で境界を整理
6. **境界カラー補正**：必要に応じて境界色を均一化
7. 「PNGで保存」で編集済み画像をダウンロード

インストールやビルドは不要。HTMLファイル単体で完結する。
